# لوحة العيادة — مزامنة مباشرة بين جميع الأجهزة

## ما تغير
- تخزين مركزي حسب التاريخ عبر Netlify Blobs.
- أي تعديل على المرضى أو الحالة أو الملاحظات يُرسل للخادم.
- جميع الصفحات المفتوحة تفحص التحديثات كل ثانيتين وتطبقها تلقائيًا.
- تحديث فوري بين تبويبات الجهاز نفسه عبر BroadcastChannel.
- البيانات تبقى بعد تحديث الموقع أو فتحه من جهاز آخر.

## طريقة النشر الصحيحة
يفضل ربط هذا المجلد بمستودع GitHub ثم نشره من Netlify حتى يتم تثبيت اعتماد @netlify/blobs وبناء Function تلقائيًا.
يمكن أيضًا رفع ZIP من صفحة Deploys إذا نفذ Netlify عملية Build للمشروع.

بعد النشر افتح نفس الرابط في جهازين، غيّر حالة مريض في جهاز، وستظهر في الجهاز الآخر خلال نحو ثانيتين.

## الخصوصية
واجهة الكتابة حاليًا متاحة لأي شخص لديه الرابط. يوصى لاحقًا بإضافة صفحة إدارة محمية أو رمز تحرير.
