# تحديث لوحة العيادة V2

التعديلات:
- الانتقال التلقائي للمريض التالي فور انتهاء وقت المريض الحالي.
- ظهور المريض التالي بحالة انتظار الموعد.
- إصلاح اختفاء قائمة تغيير الحالة؛ لم يعد الجدول يعاد بناؤه كل ثانية.
- العدادات والوقت يستمران بالتحديث كل ثانية دون إغلاق القوائم المنسدلة.

للتحديث على Netlify:
1. افتح موقعك في Netlify.
2. ادخل إلى Deploys.
3. اختر Deploy manually أو اسحب هذا المجلد/ZIP.
4. سيبقى نفس الرابط الرئيسي للموقع.
